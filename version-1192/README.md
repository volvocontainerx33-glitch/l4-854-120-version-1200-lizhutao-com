# 推荐欧美影片-热门影片片库-高清影片在线观看

这是根据上传素材生成的纯静态电影网站。

## 文件结构

- `index.html`：首页，包含 Hero 首屏轮播、推荐区、排行榜入口、分类入口。
- `categories.html`：分类总览页。
- `category/`：独立分类页，包含类型、地区、片种分类。
- `ranking.html`：排行榜页面。
- `pages/`：完整片库分页列表。
- `movie/`：影片详情页，共 2000 个。
- `assets/`：站点样式、交互脚本和 HLS 播放库。
- `data/search-index.json`：搜索索引。

## 图片说明

页面已按要求引用网站顶级目录下的 `1.jpg` 到 `150.jpg`。
若你已有这些图片，请放到网站根目录，与 `index.html` 同级。

## 数据说明

`data_2000.txt` 实际解析到 2000 部影片，已按实际数量生成。
